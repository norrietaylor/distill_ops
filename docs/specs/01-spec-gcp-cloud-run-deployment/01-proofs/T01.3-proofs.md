## Design notes

- Repo name is a constant `distillery` (held in `AR_REPO`), mirroring the
  one-repo-per-target convention. CI image references will be
  `<region>-docker.pkg.dev/<project>/distillery/distillery:<sha>`.
- `describe -> conditional create`; no blanket `|| true`. Permission or
  quota failures on describe surface as real errors.
- `--repository-format=docker` is required; `--description` is cosmetic
  but aids operators inspecting the project.
